    -   Open cmd.exe and change current directory to the directory
        where the files are located.

    -   To compile type: javac ./*.java

    -   This will compile all the files that end with .java

    -   To run type: java assignment4 ./input.txt

    -   input.txt can be replaced with any other input file name
        the user would like to use.

    -   The program will output the results to a file named output.txt.
        Open output.txt to view the results of the test.
